# 关于本站

这是 Civi 搭建的一个小型知识库，用来统一记录：

- 日常开发与运维过程中的踩坑与总结；
- 青少年犯罪预防、未成年人保护、未检/刑执实践中的经验；
- 自己在做的数字化项目（例如风险评估系统）的设计与迭代。

## 技术栈

- **框架**：VuePress v2
- **主题**：VuePress Theme Hope
- **托管**：GitHub 仓库 + Cloudflare Pages
- **部署方式**：只要在 GitHub 上提交，Cloudflare 会自动拉取代码构建并发布。

## 写作方式

- 全部使用 Markdown 编写；
- 结构上尽量「一主题一文档」，方便链接与引用；
- 重点是**记录真实的思考和实践过程**，而不是写给别人看的“完美教程”。

## 未来规划

- 持续完善技术笔记（容器、网络、数据库、自动化脚本等）；
- 梳理未检/刑执工作流，做成可视化的“工作备忘录”；
- 将青少年风险评估项目的设计思路、关键代码和数据结构整理成系统文档。
