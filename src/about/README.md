# 关于本站

这是一个用于记录「计算机系统 + 大模型技术实践」的个人 Wiki。

## 主要方向

- **系统 & 开发**  
  Linux / 网络 / Docker / 服务器环境配置与排错；
  常用命令行工具的小抄与脚本片段。

- **大模型随笔**  
  提示工程心得、调用 API 的工程实践、向量检索、RAG、工具调用等。

- **工程视角**  
  怎么把大模型融入现有系统，而不是孤立地“玩模型”。

## 技术栈

- 框架：VuePress v2
- 主题：VuePress Theme Hope
- 代码托管：GitHub
- 部署：Cloudflare Pages + 自定义域名

所有改动都通过 GitHub 网页端完成，提交后由 Cloudflare 自动构建和发布。
