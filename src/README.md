欢迎来到 **Civi 的小 Wiki**。  
这是一个完全托管在 **GitHub + Cloudflare Pages** 上的静态知识库：

- 不依赖本地开发环境，所有内容都可以在 GitHub 网页端直接编辑；
- 通过 Cloudflare 自动构建与部署，提交即发布；
- 适合长期沉淀：技术、脚本、实务经验、论文/项目笔记等。

你可以从上方导航进入不同板块，或直接从左侧侧边栏浏览内容。
